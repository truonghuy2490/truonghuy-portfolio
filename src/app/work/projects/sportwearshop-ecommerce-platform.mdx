## Overview

SportWearShop is a full-stack e-commerce platform that I built as a personal project to deepen my understanding of modern web application development and backend system design.

Unlike a tutorial project, every part of the application—from the database schema and backend architecture to the React admin dashboard—was designed and implemented by me. The platform simulates a real-world online shopping system supporting product management, inventory tracking, shopping carts, order processing, and role-based administration.

## Key Features

- **JWT Authentication** with refresh tokens and role-based authorization.
- **Product & Variant Management** supporting multiple sizes, colors, and images.
- **Three-level Category System** for flexible product organization.
- **Shopping Cart & Checkout** workflow.
- **Inventory Management** using stock movement history.
- **Cloudinary Integration** for product image uploads.
- **React Admin Dashboard** for managing products, categories, orders, and inventory.

## My Contributions

As the sole developer, I was responsible for the complete software development lifecycle—from system design and database modeling to backend development, frontend implementation, and deployment.

### Backend Development

- Designed the backend using ASP.NET Core Web API with a layered architecture.
- Implemented JWT authentication with refresh tokens and role-based authorization using ASP.NET Identity.
- Developed RESTful APIs for products, categories, brands, shopping carts, orders, inventory, and user management.
- Designed a flexible product variant system supporting multiple sizes, colors, pricing, and images.
- Built inventory tracking using stock movement history instead of directly updating quantities to improve inventory accuracy.
- Designed relational database models using SQL Server and Entity Framework Core.
- Integrated Cloudinary for secure product image storage and management.

### Frontend Development

- Built the administration dashboard using React, TypeScript, Redux Toolkit, and Vite.
- Developed management pages for products, categories, inventory, orders, and user authentication.
- Implemented reusable components, CRUD operations, filtering, pagination, and API integration.
- Connected the frontend with backend APIs while maintaining a clean separation between presentation and business logic.

## Technologies Used

- **Backend**: ASP.NET Core Web API, C#, Entity Framework Core, ASP.NET Identity
- **Database**: SQL Server
- **Authentication**: JWT Authentication, Refresh Tokens
- **Frontend**: React, TypeScript, Redux Toolkit, Vite, Tailwind CSS
- **Cloud**: Cloudinary
- **Development Tools**: Docker, Git, Swagger, Postman

## Challenges and Learnings

SportWearShop was my largest personal project and gave me the opportunity to make architectural decisions before writing code.

One of the biggest challenges was designing a flexible product model that could support multiple product variants, inventory tracking, and scalable business logic without becoming difficult to maintain.

Another challenge was implementing secure authentication using JWT access tokens and refresh tokens while keeping frontend and backend responsibilities well separated.

Building both the backend and frontend also helped me understand how different application layers communicate, making me more comfortable delivering complete end-to-end features.

Overall, this project significantly strengthened my understanding of backend architecture, API design, relational database modeling, authentication, and full-stack application development.

## Outcome

SportWearShop became my primary learning project for modern .NET development and remains the project where I explored the widest range of technologies independently.

Building the platform from scratch gave me practical experience designing software architecture, implementing secure authentication, integrating cloud services, and developing a complete full-stack application.

More importantly, the project reinforced that my strongest interest lies in backend engineering and designing maintainable, scalable software systems.
