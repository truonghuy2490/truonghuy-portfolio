## Overview

Engineering Management System (EMS) is an enterprise platform developed for JGC Holdings Corporation to centralize engineering knowledge, technical documentation, and project data.

The platform enables engineers to browse multilingual manuals, search engineering documents, and interact with an AI-powered chatbot capable of answering questions based on internal engineering documentation using Retrieval-Augmented Generation (RAG).

During my time at SPLUS Software, I worked as a Backend Developer, contributing to backend APIs, Azure cloud services, authentication, deployment, and enterprise application maintenance while collaborating within an Agile Scrum team.

## Key Features

- **Engineering Documentation Portal** for multilingual manuals.
- **Enterprise Search** across engineering documents.
- **AI-powered Chatbot** using Azure OpenAI and RAG.
- **Secure Authentication** with Microsoft Entra ID.
- **Cloud-native Architecture** built on Microsoft Azure.
- **Continuous Deployment** using Azure Pipelines.

## Engineering Workflow

One of EMS's core capabilities is synchronizing engineering data from BIM models into the enterprise management system.

Engineers create and maintain BIM models in Autodesk Revit. Through Azure-based processing pipelines, engineering metadata is extracted, validated, and synchronized into EMS, where it becomes searchable and manageable throughout the project lifecycle.
